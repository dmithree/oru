# Morning Briefing Section Rules

Read this doc when generating morning briefings or evening debriefs.

---

## Health Section
- **ONLY show if action is needed**: checkups, tests, follow-ups, appointments
- Search keywords: "next appointment", "follow-up", "konsultatsiya", "cherez", "povtor", "kontrol", "naznacheno", "rekomendovano", "recommendations / plan", "follow-ups and scheduling"
- If NO action needed: DO NOT show the Health section at all
- Habit tracking: COMPLETELY REMOVED - never show

## Therapy Section
- ONLY show if NEW results exist (sessions from today or within last 7 days)
- If NO new results: DO NOT show the Therapy section at all

## Coaching Section
- COMPLETELY REMOVED - coaching is on pause, never show
